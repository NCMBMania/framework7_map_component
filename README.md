# NCMB Map

Framework7とNCMBを組み合わせた地図コンポーネントです。

## 動画

[https://www.youtube.com/watch?v=XcmA9nmc_1g](https://www.youtube.com/watch?v=XcmA9nmc_1g)

## 機能

- 地図表示
- 地図にマーカー表示
- 現在位置に近い店舗情報をリスト表示
